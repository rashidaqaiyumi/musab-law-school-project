## 📖 Overview

Musab Practical Law School is an AI-powered Learning Management System built to simplify legal education.

The platform allows students, faculty, and administrators to access courses, manage learning resources, track progress, and collaborate through a modern and responsive web application.
## ✨ Features

- ✅ Student Dashboard
- ✅ Faculty Dashboard
- ✅ Admin Dashboard
- ✅ Course Management
- ✅ Authentication
- ✅ Responsive Design
- ✅ AI-Powered Learning Experience
